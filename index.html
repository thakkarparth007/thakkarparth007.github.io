Projects

I've mostly done small projects. Here are the links to some of those:

  • Console based calculator (C++)

    I had created this while reading "Programming: Principles and Practice Using C++". It is slightly more advanced than the code given in the book. Of course, there are bugs too.

    Warning: I had written it around 2-3 years back. I haven't gone through the code since then.

  • Another calculator (web-based)

    I had created this as a part of an induction task for the Spider club.

  • Scribblepad

    I had created this while I was in the 11th standard. I often found the need to discuss mathematics and physics problems with my friend. Doing that over telephone wasn't very convenient. Plus, I did not have access to high speed internet that I could video chat. So, I created this. A "pad" where multiple people can scribble and chat. It is not very advanced. In fact the code might even be overly complicated. I'm not maintaining the code anymore.

    My friend and I used this a lot!

  • Galaxy Destruction

    Just a simple 2 player game (both players on the same keyboard). The goal is to destroy the other space ship before it destroys you.

    Controls:
    WASD (navigation), Space (laser), Q (bullets)
    Arrow keys (navigation), Numpad0 (laser), Enter (bullets)
